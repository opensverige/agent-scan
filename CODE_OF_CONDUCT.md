# Code of Conduct

We adopt the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) verbatim.

The full text is at [contributor-covenant.org/version/2/1/code_of_conduct](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Reporting

Report Code of Conduct violations to `info@opensverige.se`. We respond within 5 business days.

Reports are confidential. Anyone in the maintainer team can be the recipient — if your concern is about a specific maintainer, address it to another maintainer instead.

## Enforcement

We follow the Contributor Covenant's enforcement ladder: correction → warning → temporary ban → permanent ban. Severity and pattern of behaviour determine which step.

Permanent bans extend across the GitHub repo and the Discord community.
