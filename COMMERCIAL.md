# Commercial licensing

The agent.opensverige source code is licensed under [FSL-1.1-MIT](LICENSE) — source-available, with a two-year non-compete clause. After two years per commit, the code auto-converts to MIT.

A commercial license waives the non-compete restriction and unlocks scenarios that the FSL forbids during the two-year window.

## you do not need a commercial license to

- run the scanner internally inside your own organisation
- use it for non-commercial education or research
- modify the code and ship those modifications back as pull requests
- offer professional services (consulting, integration, implementation) to others using the software in compliance with the FSL
- self-host an instance for your own use, including an internal company-wide tool
- fork the repository, study the code, write articles citing it, take screenshots

If any of those describe your situation, you can use the source under FSL without contacting us.

## you need a commercial license if you

1. **offer agent.opensverige.se as a managed service to third parties** — for example a "ScanSverige.se" that anyone on the internet can pay you to use, built on our source
2. **embed the scanner inside a closed-source commercial product** that competes with us — for example a security suite where this functionality is one feature among many

These two scenarios are what the FSL non-compete clause is designed to prevent. If your use case is one of them, get a commercial license from us instead.

## what a commercial license grants

- waiver of the FSL "Competing Use" restriction during the two-year window
- right to relicense modifications under your own terms (with attribution)
- written confirmation suitable for your own customers' procurement teams
- optional: priority response on security advisories that affect your deployment

It does not include hosting, support, SLA or indemnity by default. Those are separate agreements layered on top if you want them.

## hosted API tier

A separate offering from the commercial license. The hosted API at `agent.opensverige.se/api/v1` is currently in invite-only alpha. Public Hobby / Builder / Pro tiers ship in 2026 — see [docs/strategy/research/03-api-monetization-benchmarks.md](docs/strategy/research/03-api-monetization-benchmarks.md) for the planned tier structure.

You do not need a commercial license to use the hosted API — only an API key under the standard tier you sign up for.

## pricing

Commercial licenses start in the low four figures EUR per year and scale with deployment size, expected revenue derived from the scanner, and how much support you want. We don't publish a price list because the scenarios above vary too much for it to be useful.

We don't do "Enterprise" packaging. We do straightforward written agreements.
